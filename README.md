# 🖥️ System Monitor CLI

A lightweight and real-time **system monitoring tool** built with Python.
Track your system's performance directly from the terminal with a clean and minimal interface ⚡

---

## 🚀 Features

* 🧠 Live CPU usage tracking
* 💾 Real-time RAM usage monitoring
* 💽 Disk usage insights
* 🔁 Continuous refresh (live updates)
* 🧼 Clean terminal-based UI

---

## 🛠 Tech Stack

* Python 🐍
* psutil (system stats)
